## ‚ú¶ Getting Started

### Prerequisites

- Node.js v18+
- npm
- A free [Supabase](https://supabase.com) account

### 1 ‚ÄĒ Clone the repository

```bash
git clone https://github.com/seyansiuu/task-dashboard.git
cd task-dashboard
```

### 2 ‚ÄĒ Set up the database

Log into your Supabase project, open the **SQL Editor**, and run:

```sql
create table tasks (
  id uuid default gen_random_uuid() primary key,
  title text not null,
  description text,
  status text default 'Todo' check (status in ('Todo', 'In Progress', 'Completed')),
  due_date date,
  created_at timestamp default now()
);

-- Disable RLS for development
alter table tasks disable row level security;
```

### 3 ‚ÄĒ Configure the backend

```bash
cd backend
```

Create a `.env` file:

```env
SUPABASE_URL=your_supabase_project_url
SUPABASE_KEY=your_supabase_anon_key
PORT=5001
```

> Get these from **Supabase ‚Üí Project Settings ‚Üí API**

Install dependencies and start:

```bash
npm install
node server.js
# Server running on port 5001
```

### 4 ‚ÄĒ Run the frontend

Open a new terminal tab:

```bash
cd frontend
npm install
npm run dev
# App running at http://localhost:5173
```

## ‚ú¶ API Reference

Base URL: `http://localhost:5001/api`

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/tasks` | Fetch all tasks (newest first) |
| `POST` | `/tasks` | Create a new task |
| `PUT` | `/tasks/:id` | Update a task by ID |
| `DELETE` | `/tasks/:id` | Delete a task by ID |

### Example ‚ÄĒ Create a task

```bash
curl -X POST http://localhost:5001/api/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Ship the dashboard",
    "description": "Complete the HedgeOne screening project",
    "status": "In Progress",
    "due_date": "2026-05-31"
  }'
```

### Response format

```json
{
  "data": {
    "id": "uuid-here",
    "title": "Ship the dashboard",
    "description": "Complete the HedgeOne screening project",
    "status": "In Progress",
    "due_date": "2026-05-31",
    "created_at": "2026-05-27T12:00:00Z"
  }
}
```
